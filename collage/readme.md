Create a folder for each year.

Copy the index.html file from a year to the target year folder.

Adjust the year in the index.html file (2 places in the file).

Copy collage-[year].js from another year.

Adjust the year in the collage-[year].js file (2 places in the file).

Place the photos in \media\collage\[year]

Make sure node.js is installed.

From the command line run: node .\collage-[year].js
